# ChemDoodle



## Install

```html
    <!--ChemDoodle Web Components library-->
    <link rel="stylesheet" href="../../src/ChemDoodleWeb.css" type="text/css">
    <script type="text/javascript" src="../../src/ChemDoodleWeb.js"></script>

    <!-- 为了简化ChemDoodle Web 组件库，SketcherCanvas组件作为插件提供 -->
    <!--these two are required by the SketcherCanvas plugin-->
    <link rel="stylesheet" href="../../src/sketcherCanvas/jquery-ui-1.11.4.css" type="text/css">
    <script type="text/javascript" src="../../src/sketcherCanvas/ChemDoodleWeb-uis.js"></script>
```
